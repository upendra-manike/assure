# Changelog

All notable changes to Assure Testing Language will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-05

### Added
- Initial release of Assure Testing Language
- Custom CDP-based browser automation engine built from scratch
- Language parser and tokenizer for `.assure` files
- Core commands: OPEN, CLICK, TYPE, WAIT, EXPECT
- CLI runner with `.assure` file extension validation
- Support for assertions: TITLE, URL, TEXT, VISIBLE
- Cross-platform Chrome/Chromium detection
- Comprehensive documentation and examples
- MIT License

### Features
- Human-readable test syntax
- Built from scratch using Chrome DevTools Protocol
- No dependencies on Playwright or Selenium
- Lightweight and fast
- Auto-waiting for elements
- Clear error messages
